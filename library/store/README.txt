Folder for SAS stored macros
----------------------------

The creation of 'stored macros' follows the recommendations of the 2009 paper: 
"Which SASAUTOS Macros Are Available to My SAS Session?" by Droogendyk, H.
http://support.sas.com/resources/papers/proceedings09/076-2009.pdf.
See also documents sasautos_macros_storage.doroogendyk.pdf and list_sasauto.pdf in folder Z:\library\doc\.

Updating the storage library
----------------------------
Macros stored in the library include:
- macros for the calculation of RDB Xal indicators,
- ...
Original macros for the calculation of RDB Xal indicators are stored in dedicated folder Z:\Estimation\pgm\RDB.
The project createMStorageRDB.egp in folder Z:\Estimation\pgmX enables to update the stored libraries.
